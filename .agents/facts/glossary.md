## 📌 Standard Technical Terms (Remain in English in Code)

- **Access Token**: Key used to authorize API request header checks.
- **Refresh Token**: Long-term key used to renew authorization without user re-login.
- **Payload**: Core data packet transmitted in API requests or WebSockets.
- **State**: The client-side reactive store variables.
- **Guard**: Routing interceptors checking session authentication status.
